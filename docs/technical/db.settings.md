# 数据库设置字段

此文档包含部分字段的数据类型说明。

## 字段

| 字段名                         | 类型                           | 说明         |
| ------------------------------ | ------------------------------ | ------------ |
| `translate:target:language`    | `LanguageCode`                 | 翻译目标语言 |
| `translate:source:language`    | `LanguageCode`                 | 翻译源语言   |
| `translate:bidirectional:pair` | `[LanguageCode, LanguageCode]` | 双向翻译对   |
